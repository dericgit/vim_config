vim 配置
=================

这个配置是适用于网页前端开发，适用语言包括 html，css，javascript。适用系统为 windows

checkout  地址 git://github.com/hotoo/vimrc.git


使用插件
----------

* auto-pairs  括号引号补全
* bufexplorer
* FuzzyFinder 
* jsbeautify
* matchit
* mru
* neocomplcache
* snipMate
* surround
* taglist
* NERD-Commenter
* NERD-tree
* YankRing
* zencoding
